Available GitHub Tools

Your MCP server now has access to the following GitHub API tools:

Repository Management

  • List repositories
  • Create repositories
  • Get repository details
  • Manage branches

Issue Management

  • List and create issues
  • Update issue status
  • Assign users to issues
  • Manage labels

Pull Requests

  • List pull requests
  • Create pull requests
  • Review and merge PRs
  • Manage PR status

File Operations

  • Read file contents
  • Create new files
  • Browse directories
  • Commit changes

GitHub Actions

  • List workflows
  • View workflow runs
  • Monitor build status
  • Access job logs

User & Organization

  • Get user information
  • List followers/following
  • Organization access
  • Team management
